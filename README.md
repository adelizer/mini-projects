A collection of projects and demos for my YouTube channel.

## Projects

### 1. Shark Tank Egypt Dashboard

A comprehensive dashboard showcasing all startups that appeared on Shark Tank Egypt.

**Features:**
- Scrapes all Shark Tank Egypt episodes from YouTube
- Transcribes episodes using AI
- Extracts startup information (name, deal status, website, etc.)
- Interactive Next.js dashboard to browse and filter startups

**Tech Stack:**
- **Data Collection:** Python with UV package manager
- **Transcription:** OpenAI Whisper / YouTube transcripts
- **AI Extraction:** LLM-based data extraction from transcripts
- **Dashboard:** Next.js with TypeScript
